This is a repo which contains the source code for the Student_Project_Management.
To successfully run this project use the below given commands.

Prerequisites
Before running this project, you need the following software installed on your system:
Python
Git

git clone https://github.com/Keerthivarman07/Student_Project_Management

cd Student_Project_Management

python -m venv venv (to create an virtual env)

venv\Scripts\activate (activate it)

cd .\Student_Project_Management\ 

pip install -r requirements.txt (install the requirements)

python manage.py runserver

http://127.0.0.1:8000/ (open this url in a web browser to view the page)
